# Project Axiom: 博弈數值波動率與風險控制設計指南

> [!NOTE]
> 本文件詳述了 **Project Axiom** 博弈模擬分析平台中「波動率 (Volatility)」的統計學定義、計算邏輯、在「看牌百家樂 (Peek Baccarat)」特有機制下的數值表現，以及波動率對賭場風控與玩家體驗的業務意義。

---

## 1. 什麼是波動率 (Volatility)？

在博弈數學與遊戲設計中，**波動率**衡量的是玩家在單局遊戲中**實際盈虧相較於期望值 (RTP) 的偏離劇烈程度**。
* **直觀比喻**：
  * **低波動率**：像平穩前進的火車，玩家不容易大輸，也不容易暴富，資金曲線平緩（如經典百家樂主注，波動率約 `0.95`）。
  * **高波動率**：像驚險刺激的雲霄飛車，資金起伏巨大，玩家可能在短時間內資產「翻倍暴富」或「快速洗白」（如高槓桿老虎機，波動率可達 `5.0` 以上）。

---

## 2. 波動率的數學定義與計算

在 `Axiom` 數值中台（[analyzer.py](file:///../../middleware/axiom/analyzer.py)）與 C++ 模擬核心中，波動率採用**「單局淨損益之標準差 (Standard Deviation) 對初始主注本金進行基準化」**的方式進行計算：

### 2.1 計算公式
設 $N$ 為模擬總局數，第 $i$ 局的玩家淨損益為 $X_i$，初始主注金額為 $B$（本系統中固定為 $100$ 元）：

1. **單局淨損益平均值 (Mean)**：
   $$\mu = \frac{1}{N} \sum_{i=1}^{N} X_i$$

2. **單局淨損益方差 (Variance)**：
   $$\sigma^2 = \left( \frac{1}{N} \sum_{i=1}^{N} X_i^2 \right) - \mu^2$$

3. **標準差 (Standard Deviation)**：
   $$\sigma = \sqrt{\sigma^2}$$

4. **基準化波動率 (Volatility)**：
   $$\text{Volatility} = \frac{\sigma}{B}$$

> [!TIP]
> 將標準差除以初始主注 $B$ 的目的，是為了讓波動率指標**不受下注本金大小的影響**。不論玩家是押注 100 元還是 10,000 元，該遊戲的波動率常數皆維持一致，利於跨遊戲橫向評估。

## 3. 看牌百家樂的波動率分析 (實際值: 2.0160)

在固定種子 `42` 的 10 億局數值驗證中，看牌百家樂的基準化波動率精確收斂於 **`2.0160`**。

### 3.1 為什麼看牌百家樂的波動率顯著偏高？
傳統百家樂主注（押閒）的波動率僅為 **`0.95` 左右**。而看牌百家樂引入了兩大特殊機制，大幅拉寬了單局盈虧的上下限：
1. **20% 強制看牌手續費**（抽水機制）。
2. **資訊揭露後的 2x / 3x 免手續費加注**（槓桿機制）。

#### 典型損益場景對比 (以初始主注 $100$ 元為例)：

| 玩家決策與對局結果 | 傳統百家樂單局損益 | 看牌百家樂單局損益 (手續費 $20$) | 佔比說明 |
| :--- | :--- | :--- | :--- |
| **主注贏 (不加注 / Check)** | $+100$ 元 | $+80$ 元 | 平衡局 |
| **主注輸 (不加注 / Check)** | $-100$ 元 | $-120$ 元 | 平衡局 |
| **最優加注 3x 贏 (押閒且大優)** | N/A (無加注) | **$+280$ 元** (主注100 + 加注200 - 手續費20) | 玩家爆發獲利點 |
| **最優加注 3x 輸 (押閒優勢被逆轉)** | N/A (無加注) | **$-320$ 元** (主注100 + 加注200 + 手續費20) | 玩家資金大失血點 |
| **和局 (Tie)** | $0$ 元 (退回主注) | $-20$ 元 (退回主注，但手續費不退) | 磨損局 |

因為存在 **$-320$** 到 **$+280$** 這樣高達 **6 倍初始主注** 的極端單局盈虧落差，使得單局統計標準差拉高至約 $201.60$ 元，基準化波動率因而攀升至 **`2.0160`**。

## 4. 波動率在玩家體驗 (Player Experience) 的意義

> [!IMPORTANT]
> 波動率決定了遊戲的**「性格」**，影響目標玩家群體的留存率與偏好。

* **中高波動率體驗**：看牌百家樂 (`2.0160`) 屬於中高波動率類別。
* **刺激感與多巴胺釋放**：玩家資金上下起伏劇烈，能夠體驗到「一把贏回 2.8 倍主注」的爽快感，非常吸引追求刺激感與以小博大的「冒險型玩家」。
* **策略參與感**：因為加注決策是玩家主動在看牌後做出的，當贏得 3x加注對局時，玩家會將其歸功於「自己的聰明決策」，從而提升遊戲的技巧代入感與心理留存。

## 5. 波動率在賭場營運與風控 (Risk Control) 的意義

對於賭場莊家（或是遊戲開發商的營運團隊），高波動率帶來了必須嚴格監控的財務風險。

### 5.1 RTP 的收斂速度 (Confidence Interval)
波動率越高，大數法則發揮作用的速度就越慢。
下表展示了在波動率為 `2.0160` 下，不同總對局數內，玩家的**實際觀測 RTP** 落在特定區間的統計機率（以 95% 信賴區間為例）：

$$\text{RTP 波動範圍} = \text{理論 RTP} \pm 1.96 \times \frac{\text{Volatility}}{\sqrt{N}}$$

| 累積模擬局數 ($N$) | 95% 信賴區間下的觀測 RTP 範圍 | 營運端風險評估 |
| :--- | :--- | :--- |
| **10,000 局 (極速)** | $94.85\% \sim 102.75\%$ | 賭場在單日或單週內有不低的機率處於**虧損狀態** |
| **1,000,000 局 (百萬)** | $98.40\% \sim 99.20\%$ | 數值指標開始穩定，賭場已能保證穩定抽水利潤 |
| **100,000,000 局 (一億)** | $98.76\% \sim 98.84\%$ | 極度穩定，可用於合規性與公正性認證 |
| **1,000,000,000 局 (十億)** | $98.79\% \sim 98.81\%$ | 完美收斂（我們最新的測試值為 **`98.8033%`**） |

### 5.2 最大回撤 (MDD) 與賭場儲備金安全線
* **最大回撤 (Max Drawdown)**：指在模擬的時間維度中，資金曲線從歷史最高利潤點跌至歷史最低點的最大下滑幅度。
* **風險警示**：
  
  > [!CAUTION]
  > 在我們的 10 億局模擬中，最大回撤達到了 **`$2,342,722,580.00`** (約 23.4 億元，以單局主注 100 元為基準)。
  
* **風控啟示**：高波動率意味著賭場可能會在特定週期內面臨嚴重的「玩家連贏期」。賭場營運此遊戲時，**必須準備充足的營運儲備金**（通常需大於預期最大回撤的 1.5 倍），以防止短暫的資金鏈斷裂風險。

## 6. 數值設計結論與建議

1. **玩家端宣傳**：本遊戲主打「完美策略下高達 98.80% 的回報率（相較傳統百家樂押閒 98.76% 更高）」，且擁有 3 倍加注的高爆發刺激感，適合在行銷時包裝為「技巧型、高回報、高槓桿」的精品百家樂。
2. **營運風控**：上線初期應限制單局最高投注額（Table Limit），以限縮單日最大損益波動。待玩家基數與投注總額累積至百萬局規模以上後，賭場利潤將完全收斂至穩定期望值。
